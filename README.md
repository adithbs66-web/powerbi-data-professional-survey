# Data Professional Survey Dashboard - Power BI

## Project Overview
Interactive Power BI dashboard analyzing survey data from data professionals worldwide, covering salary trends, job satisfaction, and technology preferences.

## Dashboard Features
- Average salary by job title
- Happiness levels (salary & work-life balance)
- Country-wise distribution of respondents
- Favorite programming languages

## Key Insights
- Data Scientists earn the highest average salary
- Work-life satisfaction varies by role
- Python remains the most popular language
- Country of residence impacts compensation

## Tools Used
- Power BI Desktop
- Data Transformation (Power Query)
- Interactive Visualizations

## What I Learned
- Building dashboards in Power BI
- Creating DAX measures for calculations
- Power Query for data cleaning
- Designing user-friendly visuals

## Dataset
Data Professional Survey (Kaggle/Online)
